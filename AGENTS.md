This directory contains material for a course teaching beginning maths undergraduates to do some basic calculus and similar things, with computer assistance.  The original version, which was the only one that was ever used with real students, used Maple.  Since then I have made a version that uses Maxima, and another version that uses Python with sympy.

There are 11 lab sessions.  Maple instructions for those lab sessions are in labs_maple/all_labs_maple.tex, and the solutions are in labs_maple/sols/labs01.mw to labs_11.mw.  (These are XML files, but you should not need to inspect them.)  The file labs_maple/primer_maple.tex contains general information about using Maple, not tied to any particular lab session.  The file labs_maple/all_labs_maple.tex has cross-references to labs_maple/primer_maple.tex.  

Solutions for the Python versions are in labs_python/sols/lab_sols01.ipynb to labs_python/sols/lab_sols11.ipynb.  These use plotly, which I now prefer; there are also earlier versions using matplotlib in labs_python/sols_matplotlib.  These solutions are in good shape.  However, the instructions in labs_python/all_labs_python.tex, and the primer in labs_python/primer_python.tex, require more work.  They are partial translations of the Maple versions, and they are incomplete or incorrect in various ways.  The file primer_python.ipynb follows the structure of primer_python.tex and is intended as a check that everything in the primer_python.tex is correct.

Solutions for the Maxima versions are in labs_maxima/sols/lab_sols01.mac to labs_maxima/sols/lab_sols11.mac.  These are generally in good shape.  I have not yet written files labs_maxima/all_labs_maxima.tex or labs_maxima/primer_maxima.tex, but this is a lower priority than the Python versions and can be ignored for the moment.

The web directory is copied to the github.io page for this course.  There is a bash script scripts/make_docs.sh which copies the required files into the web directory.  Ideally, that should be replaced by a Python doit setup.  The file dodo_MPS224.py is for a different course; I have copied it here to be used as a loose model, although many things need to be added and subtracted.  Also, everything in the web directory should be improved so that there is a separate index page for each of the three versions, with better explanation, and automatically created HTML versions of the solution files.

Everything else can be left as it is.
